# Stakeholders

| Stakeholder         | Role in the System              | Main Need                        |
|---------------------|---------------------------------|----------------------------------|
| Aggregator (Admin)  | Primary user, logs in, manages  | Accurate orders, receipts        |
| Supplier / Farmer   | Registered in the system        | Fair record of what they sold    |
| Cooperative Manager | May view dashboard              | Daily totals and activity        |
| INES Instructor     | Evaluator                       | Clean code, working features     |
